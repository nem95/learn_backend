## 🎯 Ton premier ticket

Ouvre le ticket P1-01 :
```bash
cat phase-1/tickets/P1-01-setup-express.md
```

Ce ticket te demande de :
1. Créer un dossier `projet-1/`
2. Installer Express, TypeScript, etc.
3. Créer un serveur qui écoute sur le port 3000
4. Valider que `npm run start:dev` marche

**Durée estimée : 30-60 min**
